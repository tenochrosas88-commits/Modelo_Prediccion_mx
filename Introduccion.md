# Predicción de Goles en la Liga MX

Este proyecto consiste en el desarrollo de un modelo de predicción de goles para partidos de la Liga MX utilizando técnicas de análisis de datos y estadística. El objetivo principal es estimar la cantidad de goles que pueden anotarse en un partido a partir de información histórica y variables relacionadas con el rendimiento de los equipos.

El modelo analiza distintos factores, como estadísticas ofensivas y defensivas, desempeño reciente, condición de local o visitante y otras métricas relevantes del fútbol mexicano. A partir de estos datos, se construye un modelo capaz de identificar patrones y generar predicciones sobre el número de goles esperados en cada encuentro.

Este proyecto fue desarrollado como práctica de ciencia de datos aplicada al deporte, combinando programación, análisis estadístico y modelado predictivo. Además, busca mostrar cómo herramientas de machine learning y estadística pueden aplicarse al análisis deportivo y a la toma de decisiones basada en datos.

## Tecnologías utilizadas

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

## Objetivos

* Analizar datos históricos de partidos.
* Identificar variables importantes en la generación de goles.
* Construir un modelo predictivo.
* Evaluar el desempeño del modelo mediante métricas estadísticas.

## Resultados esperados

El proyecto busca obtener predicciones razonables sobre la cantidad de goles en partidos de la Liga MX y explorar qué factores tienen mayor influencia en el comportamiento ofensivo de los equipos.
